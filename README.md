# Prateek Kumar Pandey - Full Stack Developer Portfolio

A modern, responsive portfolio website built with React.js and Vite, showcasing my skills, projects, and professional experience as a Full Stack Developer.

## 🚀 Features

- **Responsive Design**: Optimized for all devices (mobile, tablet, desktop)
- **Modern UI/UX**: Built with Tailwind CSS and Framer Motion animations
- **Smooth Navigation**: Interactive navbar with smooth scrolling
- **Project Showcase**: Dynamic portfolio section with project details
- **Skills Section**: Comprehensive display of technical skills
- **Contact Form**: Easy way to get in touch
- **Performance Optimized**: Fast loading with Vite build tool

## 🛠️ Technologies Used

### Frontend
- **React.js** - UI framework
- **Vite** - Build tool and development server
- **Tailwind CSS** - Utility-first CSS framework
- **Framer Motion** - Animation library
- **React Router** - Client-side routing

### Development Tools
- **ESLint** - Code linting and formatting
- **Git** - Version control
- **VS Code** - Code editor

## 📦 Installation

1. Clone the repository:
```bash
git clone https://github.com/prateek8318/portfolio.git
```

2. Navigate to the project directory:
```bash
cd portfolio
```

3. Install dependencies:
```bash
npm install
```

4. Start the development server:
```bash
npm run dev
```

5. Open your browser and visit `http://localhost:5173`
